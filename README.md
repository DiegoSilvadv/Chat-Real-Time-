
#Chat em tempo real com NodeJS, Socket.io, express, ejs

#Como executar o projeto

1. Instalar o NodeJS
2. Instalar ejs express socket.io
3. yarn add ejs express socket.io (ou pode usar o npm)
4. Rodar o server.js
